# 🎂 NEXI Bake - Asistente Digital para Repostería

¡Bienvenido a NEXI Bake! Esta es una aplicación web premium diseñada para ayudar a reposteros profesionales a gestionar sus negocios de forma eficiente.

## 🚀 Características Principales:
* 📊 **Control de Inventarios:** Calcula costos exactos por gramo o mililitro con margen de merma.
* 💰 **Cotizador Inteligente:** Calcula precios finales añadiendo mano de obra y gastos indirectos.
* ☁️ **Sincronización Híbrida:** Guarda datos localmente o conéctate a NEXI Cloud.
* 💬 **Exportación a WhatsApp:** Envía presupuestos formales a clientes con un solo clic.
